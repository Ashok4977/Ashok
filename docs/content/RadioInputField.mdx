The `RadioInputField` component is used to render a labelled radio input and, optionally, associated hint text and graphic.

**Use [ChoiceFieldset](/ChoiceFieldset) to render a group of related radio input fields.** This component is only meant to be used in the case that you're building a custom radio group that is not yet supported by Primer (For example: the color mode selection in [Appearance settings](https://github.com/settings/appearance))

## Basic Example

```jsx live
<RadioInputField>
  <RadioInputField.Label>Option one</RadioInputField.Label>
  <RadioInputField.Input name="basic" value="one" />
</RadioInputField>
```

## RadioInputField

The container that handles the layout and passes the relevant IDs and ARIA attributes it's children.

`RadioInputField.Label` and `RadioInputField.Input` are required children.

### Component props

| Name       | Type                                                                                                         |     Default      | Description                                                                                          |
| :--------- | :----------------------------------------------------------------------------------------------------------- | :--------------: | :--------------------------------------------------------------------------------------------------- |
| children\* | `RadioInputField.Label`, `RadioInputField.Input`, `RadioInputField.Caption`, `RadioInputField.LeadingVisual` |        -         | Required                                                                                             |
| disabled   | `boolean`                                                                                                    |      false       | Whether the field is ready for user input                                                            |
| id         | `string`                                                                                                     | generated string | The unique identifier for this field. Used to associate the label, validation text, and caption text |

### Example: disabled

```jsx live
<RadioInputField disabled>
  <RadioInputField.Label>Option one</RadioInputField.Label>
  <RadioInputField.Input name="disabled" value="one" />
</RadioInputField>
```

## RadioInputField.Label

A `RadioInputField.Label` must be passed, but it may be visually hidden.

### Component props

| Name           | Type      | Default | Description                                 |
| :------------- | :-------- | :-----: | :------------------------------------------ |
| visuallyHidden | `boolean` |  false  | Whether the label should be visually hidden |

### Example: a visually hidden label

```jsx live
<RadioInputField>
  <RadioInputField.Label visuallyHidden>Option One</RadioInputField.Label>
  <RadioInputField.Input name="hiddenLabel" value="one" />
</RadioInputField>
```

## RadioInputField.Input

A [RadioInput](/RadioInput) will be rendered. This cannot be overridden

### Component props

The same props accepted by a [RadioInput](/RadioInput)

## RadioInputField.Caption

If this field needs additional context, a `RadioInputField.Caption` may be used to render hint text.

### Component props

| Name     | Type              | Default | Description                                                                              |
| :------- | :---------------- | :-----: | :--------------------------------------------------------------------------------------- |
| children | `React.ReactNode` |    -    | The content (usually just text) that is rendered to give contextual info about the field |

### Example

```jsx live
<RadioInputField>
  <RadioInputField.Label>Option One</RadioInputField.Label>
  <RadioInputField.Input name="withCaption" value="one" />
  <RadioInputField.Caption>Hint: the first and only option</RadioInputField.Caption>
</RadioInputField>
```

## RadioInputField.LeadingVisual

If the selectable option would be easier to understand with a visual, the `RadioInputField.LeadingVisual` component may be used.

### Component props

| Name     | Type              | Default | Description                                      |
| :------- | :---------------- | :-----: | :----------------------------------------------- |
| children | `React.ReactNode` |    -    | The visual to render before the checkbox's label |

### Example

```jsx live
<>
  <RadioInputField>
    <RadioInputField.Label>Option one</RadioInputField.Label>
    <RadioInputField.LeadingVisual>
      <MarkGithubIcon />
    </RadioInputField.LeadingVisual>
    <RadioInputField.Input name="leadingViz" value="one" />
  </RadioInputField>

  <RadioInputField>
    <RadioInputField.Label>Option two</RadioInputField.Label>
    <RadioInputField.LeadingVisual>
      <MarkGithubIcon />
    </RadioInputField.LeadingVisual>
    <RadioInputField.Input name="leadingViz" value="two" />
    <RadioInputField.Caption>This one has a caption</RadioInputField.Caption>
  </RadioInputField>
</>
```
