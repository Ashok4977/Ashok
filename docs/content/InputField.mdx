The `InputField` component is used to render a labelled text input and, optionally, associated validation text and hint text.

## Anatomy

### InputField

The container that handles the layout and passes the relevant IDs and ARIA attributes it's children.

### InputField.Label

A `InputField.Label` must be passed, but it may be visually hidden.

### InputField.Caption

If this field needs additional context, a `InputField.Caption` may be used to render hint text.

### InputField.Validation

If the field was flagged during validation, `InputField.Validation` may be used to render contextual validation information to help the user complete their task.

### Using a TextInputWithTokens input

```javascript live noinline
const TextInputWithTokensExample = () => {
  const [tokens, setTokens] = React.useState([
    {text: 'zero', id: 0},
    {text: 'one', id: 1},
    {text: 'two', id: 2}
  ])
  const onTokenRemove = tokenId => {
    setTokens(tokens.filter(token => token.id !== tokenId))
  }

  return (
    <InputField>
      <InputField.Label>Name</InputField.Label>
      <TextInputWithTokens onTokenRemove={onTokenRemove} tokens={tokens} />
    </InputField>
  )
}

render(TextInputWithTokensExample)
```

### With validation

```javascript live noinline
const ValidationExample = () => {
  const [value, setValue] = React.useState('mona lisa')
  const [validationResult, setValidationResult] = React.useState()
  const doesValueContainSpaces = inputValue => /\s/g.test(inputValue)
  const handleInputChange = e => {
    setValue(e.currentTarget.value)
  }

  React.useEffect(() => {
    if (doesValueContainSpaces(value)) {
      setValidationResult('noSpaces')
    } else if (value) {
      setValidationResult('validName')
    }
  }, [value])

  return (
    <InputField
      validationMap={{
        noSpaces: 'error',
        validName: 'success'
      }}
      validationResult={validationResult}
    >
      <InputField.Label>GitHub handle</InputField.Label>
      <TextInput as={TextInput} block value={value} onChange={handleInputChange} />
      <InputField.Validation validationKey="noSpaces">GitHub handles cannot contain spaces</InputField.Validation>
      <InputField.Validation validationKey="validName">Valid name</InputField.Validation>
      <InputField.Caption>With or without "@". For example "monalisa" or "@monalisa"</InputField.Caption>
    </InputField>
  )
}

render(ValidationExample)
```

## Props

### InputField

| Name             | Type                                                                                                                  |     Default      | Description                                                                                                                                                                                                          |
| :--------------- | :-------------------------------------------------------------------------------------------------------------------- | :--------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| children\*       | `InputField.Label`, `InputField.Caption`, `InputField.Validation`, `Autocomplete`, `TextInput`, `TextInputWithTokens` |        -         | Required                                                                                                                                                                                                             |
| disabled         | `boolean`                                                                                                             |      false       | Whether the field is ready for user input                                                                                                                                                                            |
| id               | `string`                                                                                                              | generated string | The unique identifier for this field. Used to associate the label, validation text, and caption text                                                                                                                 |
| required         | `boolean`                                                                                                             |      false       | Whether this field must have a value for the user to complete their task                                                                                                                                             |
| validationMap    | `Record<string, 'error'\|'sucess'>`                                                                                   |        -         | A map of validation statuses and their associated validation keys. When one of the validation keys is passed to the `validationResult` prop, the associated validation message will be rendered in the correct style |
| validationResult | `string`                                                                                                              |        -         | The key of the validation message to show                                                                                                                                                                            |

### InputField.Label

| Name           | Type      | Default | Description                                 |
| :------------- | :-------- | :-----: | :------------------------------------------ |
| visuallyHidden | `boolean` |  false  | Whether the label should be visually hidden |

### InputField.Caption

| Name     | Type              | Default | Description                                                                              |
| :------- | :---------------- | :-----: | :--------------------------------------------------------------------------------------- |
| children | `React.ReactNode` |    -    | The content (usually just text) that is rendered to give contextual info about the field |

### InputField.Validation

| Name          | Type              | Default | Description                                                                                                                                                                                                       |
| :------------ | :---------------- | :-----: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| children      | `React.ReactNode` |    -    | The content (usually just text) that is rendered to give contextual info about the validation result for the field                                                                                                |
| validationKey | `string`          |    -    | The key of the property from `InputField` that corresponds to this validation message. When `InputField`'s `validationResult` prop matches `InputField.Validation`'s `validationKey` prop, this message is shown. |
