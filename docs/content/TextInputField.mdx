import {TextInputField, TextInputWithTokens} from '@primer/components'

The `TextInputField` component is used to render a labelled text input and, optionally, associated validation text and hint text.

## Basic Example

```jsx live
<TextInputField>
  <TextInputField.Label>Name</TextInputField.Label>
  <TextInputField.Input />
</TextInputField>
```

## TextInputField

The container that handles the layout and passes the relevant IDs and ARIA attributes it's children.

`TextInputField.Label` and `TextInputField.Input` are required children.

### Component props

| Name             | Type                                                                                                  |     Default      | Description                                                                                                                                                                                                          |
| :--------------- | :---------------------------------------------------------------------------------------------------- | :--------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| children\*       | `TextInputField.Label`, `TextInputField.Input`, `TextInputField.Caption`, `TextInputField.Validation` |        -         | Required                                                                                                                                                                                                             |
| disabled         | `boolean`                                                                                             |      false       | Whether the field is ready for user input                                                                                                                                                                            |
| id               | `string`                                                                                              | generated string | The unique identifier for this field. Used to associate the label, validation text, and caption text                                                                                                                 |
| required         | `boolean`                                                                                             |      false       | Whether this field must have a value for the user to complete their task                                                                                                                                             |
| validationMap    | `Record<string, 'error'\|'sucess'>`                                                                   |        -         | A map of validation statuses and their associated validation keys. When one of the validation keys is passed to the `validationResult` prop, the associated validation message will be rendered in the correct style |
| validationResult | `string`                                                                                              |        -         | The key of the validation message to show                                                                                                                                                                            |

### Example: required

```jsx live
<TextInputField required>
  <TextInputField.Label>Name</TextInputField.Label>
  <TextInputField.Input />
</TextInputField>
```

### Example: disabled

```jsx live
<TextInputField disabled>
  <TextInputField.Label>Name</TextInputField.Label>
  <TextInputField.Input />
</TextInputField>
```

## TextInputField.Label

A `TextInputField.Label` must be passed, but it may be visually hidden.

### Component props

| Name           | Type      | Default | Description                                 |
| :------------- | :-------- | :-----: | :------------------------------------------ |
| visuallyHidden | `boolean` |  false  | Whether the label should be visually hidden |

### Example: a visually hidden label

```jsx live
<TextInputField>
  <TextInputField.Label visuallyHidden>Name</TextInputField.Label>
  <TextInputField.Input />
</TextInputField>
```

## TextInputField.Input

By default, a [TextInput](/TextInput) will be rendered. You may override this and render another text input such as an [Autocomplete](/Autocomplete) or a [TextInputWithTokens](/TextInputWithTokens).

### Component props

| Name | Type                             |  Default  | Description                   |
| :--- | :------------------------------- | :-------: | :---------------------------- |
| as   | `string`\|`React.ReactComponent` | TextInput | The input component to render |

The other props will be the same as the component that is passed to the `as` prop.

### Example: using a TextInputWithTokens

```javascript live noinline
const TextInputWithTokensExample = () => {
  const [tokens, setTokens] = React.useState([
    {text: 'zero', id: 0},
    {text: 'one', id: 1},
    {text: 'two', id: 2}
  ])
  const onTokenRemove = tokenId => {
    setTokens(tokens.filter(token => token.id !== tokenId))
  }

  return (
    <TextInputField>
      <TextInputField.Label>Name</TextInputField.Label>
      <TextInputField.Input as={TextInputWithTokens} onTokenRemove={onTokenRemove} tokens={tokens} />
    </TextInputField>
  )
}

render(TextInputWithTokensExample)
```

## TextInputField.Caption

If this field needs additional context, a `TextInputField.Caption` may be used to render hint text.

### Component props

| Name     | Type              | Default | Description                                                                              |
| :------- | :---------------- | :-----: | :--------------------------------------------------------------------------------------- |
| children | `React.ReactNode` |    -    | The content (usually just text) that is rendered to give contextual info about the field |

### Example

```jsx live
<TextInputField>
  <TextInputField.Label>Name</TextInputField.Label>
  <TextInputField.Input />
  <TextInputField.Caption>Hint: your first name</TextInputField.Caption>
</TextInputField>
```

## TextInputField.Validation

If the field was flagged during validation, `TextInputField.Validation` may be used to render contextual validation information to help the user complete their task.

### Component props

| Name          | Type              | Default | Description                                                                                                                                                                                                                   |
| :------------ | :---------------- | :-----: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| children      | `React.ReactNode` |    -    | The content (usually just text) that is rendered to give contextual info about the validation result for the field                                                                                                            |
| validationKey | `string`          |    -    | The key of the property from `TextInputField` that corresponds to this validation message. When `TextInputField`'s `validationResult` prop matches `TextInputField.Validation`'s `validationKey` prop, this message is shown. |

### Example

```javascript live noinline
const ValidationExample = () => {
  const [value, setValue] = React.useState('mona lisa')
  const [validationResult, setValidationResult] = React.useState()
  const doesValueContainSpaces = inputValue => /\s/g.test(inputValue)
  const handleInputChange = e => {
    setValue(e.currentTarget.value)
  }

  React.useEffect(() => {
    if (doesValueContainSpaces(value)) {
      setValidationResult('noSpaces')
    } else if (value) {
      setValidationResult('validName')
    }
  }, [value])

  return (
    <TextInputField
      validationMap={{
        noSpaces: 'error',
        validName: 'success'
      }}
      validationResult={validationResult}
    >
      <TextInputField.Label>GitHub handle</TextInputField.Label>
      <TextInputField.Input as={TextInput} block value={value} onChange={handleInputChange} />
      <TextInputField.Validation validationKey="noSpaces">
        GitHub handles cannot contain spaces
      </TextInputField.Validation>
      <TextInputField.Validation validationKey="validName">Valid name</TextInputField.Validation>
      <TextInputField.Caption>With or without "@". For example "monalisa" or "@monalisa"</TextInputField.Caption>
    </TextInputField>
  )
}

render(ValidationExample)
```
